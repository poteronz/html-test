git init ( создание репозитория ) ( 1 раз для 1 проекта )
git add * ( Добавление файлов для отслеживания изменений ) ( каждый раз при создании новой версии )
git commit -m "Comment" ( Создание версии ) ( каждый раз при создании новой версии )
git push ( Отправка созданной версии на удалённый репозиторий ) ( каждый раз при создании новой версии/необязательно )
